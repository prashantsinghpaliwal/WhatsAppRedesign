# WhatsApp Redesign

**WhatsApp Redesign** is a Material Redesign for WhatsApp App. It has been built with *Latest Android development* techniques. Feel free to use it in your projects.

## About
It has two screens - **Chat Screen** & **Calls Screen**. Both of these screens are basically fragments in an activity with **Stories List** presnt in a **CollapsingToolbarLayout**. 

- Showcases redesigning of **WhatsApp**. 
- Much easier access to stories.

**Postman**'s Mock Server has been used for creating mock api for chats, calls & stories.
